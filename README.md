# UserMap

Standalone workspace for capturing, organizing, and exporting reusable context.

## What It Includes
- Account sign-in and recovery flows
- UserMap tree editing
- Storyboard and filtered context views
- Prompt and JSON export controls
- Context import and server-backed consolidation

## Required Environment Variables
- `VITE_AUTH_POOL_ID`
- `VITE_AUTH_CLIENT_ID`
- `VITE_API_BASE_URL`

`.env.example` uses placeholders only. Supply your own values before running locally.

## Development
```bash
npm install
npm run dev
```
